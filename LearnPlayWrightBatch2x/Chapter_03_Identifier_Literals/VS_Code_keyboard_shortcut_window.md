# VS Code Keyboard Shortcuts — Windows

A concise reference of commonly used Visual Studio Code keyboard shortcuts for Windows (modifiers: Ctrl, Alt, Shift, Windows key = ⊞).

## Basics
- Ctrl+P : Quick Open (Go to File)
- Ctrl+N : New File
- Ctrl+O : Open File
- Ctrl+S : Save
- Ctrl+W : Close Editor
- Alt+F4 : Quit VS Code

## Editing
- Ctrl+X : Cut line (when no selection)
- Ctrl+C : Copy
- Ctrl+V : Paste
- Ctrl+Z : Undo
- Ctrl+Y or Ctrl+Shift+Z : Redo
- Shift+Alt+F : Format Document
- Ctrl+/ : Toggle line comment
- Shift+Alt+A : Toggle block comment
- Alt+Up / Alt+Down : Move line up/down
- Ctrl+D : Add selection to next find match (multi-cursor)
- Shift+Alt+I : Insert cursor at end of each line selected (column selection)
- Ctrl+L : Select current line

## Navigation
- Ctrl+T : Show all symbols
- Ctrl+G : Go to Line
- Ctrl+F : Find
- Ctrl+Shift+F : Search across files
- Ctrl+B : Toggle Sidebar visibility
- Ctrl+\ : Split editor
- Ctrl+1 / Ctrl+2 / Ctrl+3 : Focus into 1st/2nd/3rd editor group
- Ctrl+Tab : Switch to next editor (MRU)
- Ctrl+Shift+E : Show Explorer / Focus
- Ctrl+Shift+D : Show Debug view
- Ctrl+Shift+G : Show Source Control view

## Files & Tabs
- Ctrl+K Ctrl+S : Keyboard Shortcuts
- Ctrl+K Ctrl+W : Close all editors
- Ctrl+K Z : Zen Mode
- Ctrl+Shift+T : Reopen closed editor

## Terminal
- Ctrl+` : Toggle integrated terminal
- Ctrl+K : Clear terminal (may vary)
- Ctrl+Shift+C : Open new terminal (or copy path)

## Debugging
- F5 : Start/Continue
- Shift+F5 : Stop
- F9 : Toggle breakpoint
- F10 : Step Over
- F11 : Step Into
- Shift+F11 : Step Out

## Multi-cursor & Selection
- Ctrl+Click : Insert cursor
- Alt+Click and drag : Column (box) selection
- Ctrl+Shift+L / Ctrl+L : Select line / Select all occurrences of current selection

## Refactor & Intellisense
- Ctrl+. : Quick Fix / Show Code Actions
- Ctrl+Space : Trigger Suggest (IntelliSense)
- Ctrl+Shift+O : Go to Symbol in File

## Source Control (Git)
- Ctrl+Shift+G : Open Source Control
- Ctrl+Enter (in commit message) : Commit (depends on setup)

## Useful Command Palette
- Ctrl+Shift+P or F1 : Show Command Palette

## Notes
- Some shortcuts depend on installed extensions or user keybindings.
- To view or customize shortcuts, open `Preferences: Open Keyboard Shortcuts` (Ctrl+K Ctrl+S).
