# VS Code Keyboard Shortcuts — macOS

A concise reference of commonly used Visual Studio Code keyboard shortcuts for macOS (macOS modifiers: Command = ⌘, Option = ⌥, Control = ⌃, Shift = ⇧).

## Basics
- ⌘P : Quick Open (Go to File)
- ⌘N : New File
- ⌘O : Open File
- ⌘S : Save
- ⌘W : Close Editor
- ⌘Q : Quit VS Code

## Editing
- ⌘X : Cut line (when no selection)
- ⌘C : Copy
- ⌘V : Paste
- ⌘Z : Undo
- ⇧⌘Z or ⌘Y : Redo
- ⇧⌥F or ⇧⌘F : Format Document (command varies by extension)
- ⌘/ : Toggle line comment
- ⇧⌘A : Toggle block comment
- ⌥↑ / ⌥↓ : Move line up/down
- ⌘D : Add selection to next find match (multi-cursor)
- ⌥⌘I : Insert cursor at end of each line selected (column selection)
- ⌘L : Select current line

## Navigation
- ⌘T : Show all symbols
- ⌘G : Go to Line (press ⌘G then line number)
- ⌘F : Find
- ⇧⌘F : Search across files
- ⌘B : Toggle Sidebar visibility
- ⌘\ : Split editor
- ⌘1 / ⌘2 / ⌘3 : Focus into 1st/2nd/3rd editor group
- ⌃Tab : Switch to next editor (MRU)
- ⇧⌘E : Show Explorer / Focus
- ⇧⌘D : Show Debug view
- ⇧⌘G : Show Source Control view

## Files & Tabs
- ⌘K ⌘S : Keyboard Shortcuts
- ⌘K ⌘W : Close all editors
- ⌘K Z : Zen Mode
- ⇧⌘T : Reopen closed editor

## Terminal
- ⌃` : Toggle integrated terminal
- ⌘K : Clear terminal (may vary)
- ⇧⌘C : Open new terminal (command palette/shortcut may vary)

## Debugging
- F5 : Start/Continue
- ⇧F5 : Stop
- F9 : Toggle breakpoint
- F10 : Step Over
- F11 : Step Into
- ⇧F11 : Step Out

## Multi-cursor & Selection
- ⌘Click : Insert cursor
- ⌥Click and drag : Column (box) selection
- ⌘ShiftL / ⌘L : Select line / Select all occurrences of current selection

## Refactor & Intellisense
- ⌘.` : Quick Fix / Show Code Actions
- ⌃Space : Trigger Suggest (IntelliSense)
- ⇧⌘O : Go to Symbol in File

## Source Control (Git)
- ⇧⌘G : Open Source Control
- ⌘Enter (in commit message) : Commit (depends on setup)

## Useful Command Palette
- ⇧⌘P or F1 : Show Command Palette

## Notes
- Some shortcuts depend on installed extensions or user keybindings.
- To view or customize shortcuts, open `Preferences: Open Keyboard Shortcuts` (⇧⌘P then type "Keyboard Shortcuts").
